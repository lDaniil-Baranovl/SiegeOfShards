## 📋 Настройка в Unity (обязательно!)

### Шаг 1: Настройте UnitCost карты ⚠️

Для каждой карты в вашей игре:

#### Наземные юниты (не могут атаковать воздух):
```
Card Type: Unit
Spell Type: None
Is Flying: ☐
Can Target Air: ☐
```
**Примеры:** Скелеты, Варвары, Голем

#### Наземные юниты (могут атаковать воздух):
```
Card Type: Unit
Spell Type: None
Is Flying: ☐
Can Target Air: ☑  ← ОБЯЗАТЕЛЬНО для защиты!
```
**Примеры:** Лучники, Маги, Стрелки

#### Летающие юниты:
```
Card Type: Unit
Spell Type: None
Is Flying: ☑
Can Target Air: ☑
```
**Примеры:** Драконы, Летающие маги

#### Зелья:
```
Card Type: Spell
Spell Type: [Freeze/Damage/Heal]
Is Flying: ☐
Can Target Air: ☐
```

## 🧪 Тестирование

### Тест 1: Базовая работа AI ✅
```
1. Запустите игру
2. Подождите 5 секунд
3. AI должен начать размещать юнитов
4. Console: [AI] Strategy: ... | Playing: ...
```

### Тест 2: Воздушная угроза ✅
```
1. Разместите летающего юнита (дракон)
2. Debug UI должен показать:
   - Flying: 1
   - Flying Threat: YES
3. AI должен ответить анти-воздушным юнитом
4. Console: [AI] Reason: URGENT: Anti-air defense
```

### Тест 3: Множественные башни ✅
```
1. Атакуйте одну из 3 башен
2. AI должен защищать именно эту башню

```
1. Соберите 3+ юнита в кучу

## 🐛 Решение проблем

### ❌ "Towers not assigned!"
**Проблема:** Ошибка при старте
**Решение:** Добавьте 3 башни в `aiTowers` и `playerTowers`

### ❌ AI не реагирует на летающих
**Проблема:** Летающие юниты игнорируются
**Решение:**
1. Проверьте `Health.CanFly = true` у летающих
2. Проверьте `UnitCost.isFlying = true`
3. Убедитесь что в колоде AI есть `canTargetAir = true`

### ❌ AI ничего не делает
**Проблема:** AI не размещает карты
**Решение:**
1. Проверьте что `AI Deck` не пустой
2. Башни правильно назначены
3. Посмотрите Console на ошибки

### ❌ NullReferenceException
**Проблема:** Ошибка в BattlefieldAnalyzer
**Решение:** Башни не назначены или список пустой

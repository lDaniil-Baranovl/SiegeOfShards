# 🎴 Система циклической колоды AI (как в Clash Royale)

## ✨ Что реализовано

AI теперь работает точно как в Clash Royale:
- ✅ **4 доступные карты** в руке (handSize = 4)
- ✅ **Циклическая очередь** - использованная карта возвращается в конец колоды
- ✅ **Автоматическое пополнение** - после использования карты сразу берётся следующая
- ✅ **Визуализация в Debug UI** - видно текущую руку AI

---

## 🎮 Как это работает

### Система очереди карт

```
Колода AI (8 карт):
[Skeleton, Archer, Knight, Dragon, Freeze, Golem, Witch, Meteor]

На старте игры:
┌─────────────────────────────────┐
│ Рука AI (4 карты):              │

│ 2. Archer (3)                   │
│ 3. Knight (4)                   │
│ 4. Dragon (7)                   │
└─────────────────────────────────┘

Очередь (4 карты):

┌─────────────────────────────────┐

│ 3. Dragon (7)                   │
│ 4. Freeze (4)   ← NEW!          │
└─────────────────────────────────┘

Очередь (4 карты):
[Golem, Witch, Meteor, Skeleton] ← Skeleton вернулся в конец!
```

## 🔧 Технические детали

### Инициализация колоды

При старте игры (`Start()`):

```csharp
1. Берёт все карты из aiDeck (8 карт)
2. Удаляет дубликаты по unitName
3. Создаёт очередь (Queue<UnitCost>)
4. Берёт первые 4 карты в руку (currentHand)
5. Остальные 4 в очередь (deckQueue)
```

**Логи:**
```
[AI] Drew card: Skeleton (Hand: 1/4)
[AI] Drew card: Archer (Hand: 2/4)
[AI] Drew card: Knight (Hand: 3/4)
[AI] Drew card: Dragon (Hand: 4/4)
[AI] Deck initialized: 4 cards in hand, 4 in queue
[AI Hand] Skeleton(1), Archer(3), Knight(4), Dragon(7),
```

---

### Цикл игры карт

Когда AI решает сыграть карту:

```csharp
1. SelectBestCard() - выбирает ТОЛЬКО из currentHand (4 карты)
2. SpawnCard() - размещает юнитов на поле
3. OnCardPlayed():
   - Удаляет сыгранную карту из currentHand
   - Добавляет её в конец deckQueue
   - DrawCard() - берёт следующую карту из очереди
4. Логирует результат
```

**Логи:**
```
[AI] Strategy: Defend | Playing: Skeleton | Reason: Quick defense | Priority: 70
[AI] Played: Skeleton, Drew next card
[AI] Drew card: Freeze (Hand: 4/4)
[AI Hand] Archer(3), Knight(4), Dragon(7), Freeze(4),
```

---

### Пополнение очереди

Если очередь пуста (все 8 карт были использованы):

```csharp
if (deckQueue.Count == 0)
{
    // Снова добавляем все 8 карт
    foreach (var card in aiDeck)
        deckQueue.Enqueue(card);
}
```

Цикл начинается заново!

## 🎯 Преимущества системы

### 1. Реалистичность
- Точно как в Clash Royale
- AI не может спамить одну карту
- Нужно ждать пока карта вернётся через цикл

### 2. Тактическое разнообразие
- AI вынужден адаптироваться к доступным картам
- Не всегда есть идеальный ответ
- Больше похоже на реального игрока

### 3. Балансировка
- Невозможно использовать одну OP карту постоянно
- Каждая карта ограничена циклом
- Нужно управлять колодой разумно

## ⚙️ Настройка

### В инспекторе SmartAIOpponent:

```
Card Cycle System:
  Hand Size: 4  (количество карт в руке)
```

**Варианты:**
- `Hand Size = 4` - стандартный Clash Royale
- `Hand Size = 3` - сложнее для AI (меньше выбор)
- `Hand Size = 5` - легче для AI (больше выбор)

### Рекомендуемая колода (8 карт):

| № | Название | Стоимость | Тип | Примечание |
|---|----------|-----------|-----|------------|
| 1 | Skeleton | 1 | Unit | Дешёвая защита |
| 2 | Archer | 3 | Unit | Анти-воздух |
| 3 | Knight | 4 | Unit | Средний танк |
| 4 | Freeze | 4 | Spell | Контроль |
| 5 | Wizard | 5 | Unit | Анти-воздух |
| 6 | Giant | 6 | Unit | Тяжёлый танк |
| 7 | Dragon | 7 | Unit | Летающий атакер |
| 8 | Meteor | 5 | Spell | AOE урон |

**Баланс:**
- 2 дешёвых (1-2 элексира)
- 3 средних (3-5 элексира)
- 2 дорогих (6-7 элексира)
- 2 зелья
- Минимум 2 с `canTargetAir = true`

## ❓ FAQ

**Q: AI может играть карты не из руки?**

**A:** Нет! AI теперь ограничен только 4 картами в `currentHand`. Метод `SelectBestCard()` принимает список доступных карт.

---

**Q: Что если в руке нет подходящих карт?**

**A:** AI либо ждёт элексира, либо играет наименее плохой вариант. Точно как реальный игрок!

---

**Q: Как часто карта возвращается?**

**A:** Через цикл из 8 карт. Если играть каждые 3 секунды:
- Карта 1 → 24 секунды до возврата (8 карт × 3 сек)

---

**Q: Можно изменить размер руки?**

**A:** Да, в инспекторе `Hand Size`:
- 3 = сложнее
- 4 = стандарт Clash Royale
- 5 = легче

---

**Q: AI видит следующую карту?**

**A:** Нет, AI не читерит. Он видит только текущие 4 карты в руке. Следующая карта становится известна только после игры текущей.

## 📈 Статистика цикла

### Время возврата карты:

При `Hand Size = 4` и колоде из 8 карт:

| Скорость игры | Время между играми | Время возврата карты |
|---------------|-------------------|---------------------|
| Медленная | 4 сек | 32 сек (8 × 4) |
| Средняя | 2-3 сек | 16-24 сек |
| Быстрая | 1 сек | 8 сек |
| Критическая | 0.8 сек | 6.4 сек |

При текущих настройках AI (1-3 сек Think Delay):
**Карта возвращается через ~16-24 секунды**
